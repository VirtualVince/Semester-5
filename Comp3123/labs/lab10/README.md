## Description
A React application with a data entry form that collects user information and displays it upon submission.

## Features
- Email input
- Full Name input
- Address fields
- City, Province, and Postal Code
- Terms and Conditions checkbox
- Displays submitted data in a formatted table

## How to Run
1. Install dependencies:
```
   npm install
```

2. Start the application:
```
   npm start
```

3. Open browser at `http://localhost:3000`

## Technologies Used
- React.js
- CSS
